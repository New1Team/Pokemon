### 💡 주요 개선 사항 요약
* **데이터 정제:** 인물명 혼용 문제를 해결하기 위해 한글 매핑 로직 및 프롬프트 제약 조건 강화
* **성능 최적화:** 대용량 데이터(500+ 에피소드) 처리를 위해 동기 방식을 비동기(`async`) 처리 방식으로 전환하여 시간 단축
* **자동화 로직:** 노드 수동 관리의 번거로움을 AI 자동 감지 로직으로 대체하여 데이터 확장성 확보
